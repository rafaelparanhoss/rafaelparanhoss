## Projetos em destaque

### [Milho Safrinha RF vs U-Net](https://github.com/rafaelparanhoss/milho-safrinha-rf-vs-unet)
Comparação reprodutível entre **Random Forest no Google Earth Engine** e **U-Net** para mapeamento de milho safrinha em Mato Grosso com séries temporais Landsat 8/9.

### [INPE Queimadas ETL](https://github.com/rafaelparanhoss/inpe-queimadas-etl)
Projeto end-to-end com **ETL geoespacial + PostGIS + FastAPI + aplicação web** para ingestão diária, enriquecimento espacial e exploração interativa de focos de queimadas do INPE.
