## Projetos em destaque

### [Milho Safrinha — U-Net Multissensor](https://github.com/rafaelparanhoss/milho-safrinha-mt-unet-multissensor)

Mapeamento de milho 2ª safra em Mato Grosso com **U-Net 2D**, combinando **Landsat 8/9**, **Sentinel-2**, **Sentinel-1 SAR**, bandas de observações válidas e pós-processamento espacial.

**Principais pontos:**

* cubo multissensor com 25 canais;
* exportação dos dados no Google Earth Engine;
* treinamento e inferência em Python;
* segmentação binária milho / não milho;
* comparação agregada com IBGE/PAM 2023;
* documentação, figuras e pipeline reprodutível.

---

### [Milho Safrinha RF vs U-Net](https://github.com/rafaelparanhoss/milho-safrinha-rf-vs-unet)

Comparação entre **Random Forest no Google Earth Engine** e **U-Net** para mapeamento de milho safrinha em Mato Grosso com mosaicos sazonais Landsat 8/9.

**Principais pontos:**

* classificação supervisionada com Random Forest;
* segmentação semântica com U-Net;
* uso de amostras de referência;
* avaliação de métricas e área mapeada;
* base metodológica associada à dissertação de mestrado.

---

### [INPE Queimadas ETL](https://github.com/rafaelparanhoss/inpe-queimadas-etl)

Projeto end-to-end para ingestão, processamento e disponibilização de focos de queimadas do INPE.

**Principais pontos:**

* ETL geoespacial em Python;
* armazenamento em PostGIS;
* enriquecimento espacial dos dados;
* API com FastAPI;
* aplicação web para exploração interativa;
* pipeline automatizado e reprodutível.
